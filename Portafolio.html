Proyecto JIEM

Hace unos meses, decidí crear una landing page para un taller de confección llamado JIEM. Mi objetivo era presentar los productos más destacados del negocio y proporcionar una forma sencilla para que los visitantes se conectaran con nosotros.

¿Quieres algo único? En JIEM, no solo ofrecemos productos listos para usar, sino también la posibilidad de personalizar tus prendas. Nuestra sección de cotización te permite elegir telas, colores y estilos para crear la prenda perfecta. Imagina una chaqueta a medida que se adapte a tu estilo o una falda que refleje tu personalidad. ¡En JIEM, hacemos tus sueños realidad!

Comencé construyendo la estructura básica utilizando HTML. Luego, apliqué estilos elegantes con CSS para dar vida a la página.Bootstrap fue mi aliado para la creación de diseños responsivos y componentes reutilizables. Los grids y las clases predefinidas de Bootstrap agilizaron mi trabajo.
